# สรุปการเรียน Web Design 3 สัปดาห์แรก

## สัปดาห์ที่ 1
- เริ่มต้นทำหน้า HTML แบบพื้นฐานในไฟล์ `WebDesignW1/index.html`
- ฝึกสร้างโครงสร้างหน้าเว็บด้วย `<!DOCTYPE html>`
- ใช้แท็กพื้นฐาน เช่น `<html>`, `<head>`, `<body>`, `<h1>`, `<h2>`
- ทดลองเขียนหัวข้อและข้อความง่าย ๆ เพื่อเข้าใจโครงสร้างของ HTML

## สัปดาห์ที่ 2
- ทำไฟล์ `WebDesignW2/index.html` เป็นหน้า Tutorial HTML ที่มีเนื้อหาอธิบายหลักการใช้งาน
- ฝึกใช้แท็ก `<ul>`, `<li>`, `<a>` เพื่อสร้างเมนูลิงก์ในหน้าเดียวกัน
- เขียนเนื้อหาเกี่ยวกับหัวข้อ HTML พื้นฐาน เช่น Heading, Paragraph, Links, Images, Lists, Tables, Forms, Multimedia และ Semantic HTML
- ใช้ `<style>` ใน `<head>` เพื่อเขียน CSS เล็กน้อย เช่น การกำหนดฟอนต์, สีพื้นหลัง, สีหัวข้อ, สไตล์ลิงก์ และกล่องเนื้อหา
- ทำลิงก์ภายในหน้า (`anchor link`) เพื่อให้กลับไปยังส่วนบนของหน้าได้สะดวก
- สร้างบล็อก Workshop และลิงก์ไปยังเว็บภายนอกสำหรับเรียนต่อ

## สัปดาห์ที่ 3
- สร้างไฟล์ `WebdesignW3/W03-heading.html` เป็นงาน Workshop เรื่อง Heading
- ฝึกจัดหน้าเว็บด้วย CSS เช่น `box-sizing`, `margin`, `padding`, `background-color`, `box-shadow` และ `border-radius`
- ออกแบบโครงสร้างหน้าด้วย `header`, `nav`, `section`, `div` และคลาสของ CSS
- ทดลองแสดงตัวอย่างหัวข้อ `h1`–`h6` พร้อมคำอธิบายการใช้งานแต่ละระดับ
- ใช้สไตล์ป้ายกำกับสีต่าง ๆ เพื่อตัวอย่างความแตกต่างของ Heading แต่ละระดับ
- ฝึกเขียนคำอธิบายเป็นภาษาไทยและจัดรูปแบบหน้าให้ดูเป็นระบบ

## สัปดาห์ที่ 4
- เริ่มทำงานกับไฟล์ `WebDesignW4/W04-01-formmating.html` และ `css/W04-formatting.css`
- เรียนรู้การจัดรูปแบบข้อความด้วยแท็ก HTML เช่น `<b>`, `<strong>`, `<i>`, `<em>`, `<u>`, `<mark>`, `<small>`, `<del>`, `<sup>` และ `<sub>`
- ฝึกสร้างหน้าเว็บที่มีส่วนประกอบแบบ card, hero section, tip box และ footer เพื่อให้หน้าเว็บดูเรียบร้อยขึ้น
- เรียนรู้การใช้ CSS สำหรับจัด layout ด้วย Flexbox และ Grid
- ฝึกปรับแต่งสีพื้นหลัง สีข้อความ ขอบกรอบ และการเว้นระยะห่างให้สวยงาม
- เรียนรู้การแยกไฟล์ HTML กับ CSS ให้อยู่คนละไฟล์ เพื่อให้โค้ดดูเป็นระเบียบและจัดการง่ายขึ้น
- ทำความเข้าใจการใช้ `pre` สำหรับแสดงโค้ดตัวอย่าง และ `code` สำหรับเน้นคำสั่งในข้อความ
